Notificaciones
****************
Módulo del sistema que se encarga de la gestión de Notificaciones.

Las notificaciones son enviadas a los usuarios por correo electrónico, así tambien, se muestra
la lista de notificaciones pendientes dentro de la página de inicio del sistema.
Una vez el usuario lee la notificación dentro del sistema esta quedará como leida.
