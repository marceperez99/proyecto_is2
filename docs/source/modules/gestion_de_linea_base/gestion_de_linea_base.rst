Lineas Base
*************
Módulo del sistema que se encarga de la gestión de las lineas base dentro de una fase.

Una linea base es una descripción acordada de los atributos de un conjunto de items en un momento dado. Tambien es un requerimiento para empezar a crear items en una fase posterior a la actual.

Si se desea modificar algun item dentro de una linea base es necesario generar una solicitud de rompimiento de Linea Base.
