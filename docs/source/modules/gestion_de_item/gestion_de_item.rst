Ítems
*************
Módulo encargado de la administración de los items pertenecientes a una fase.

Un usuario con el permiso de crear items dentro de una fase del proyecto puede seleccionar un tipo de item
perteneciente a una fase y utilizarlo como una plantilla para sus atributos.
